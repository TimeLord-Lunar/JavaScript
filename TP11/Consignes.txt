suivantes :

departementNaissance string

libelleRue string

• Démarrez le serveur json-server avec le fichier utilisateurs.json comme source de
données
• Dupliquez le fichier tp10_formulaire.html afin de créer un
tp11_formulaire_json_server.html

digicode.
